About this Demo

Catalogue Manager



How does this demo work?


This sample Node.js application that uses the invoke API from the ibm-cics-api module to call COBOL programs included in the CICS catalog manager. The Node.js application includes the following elements:

  • Browser front end web site that uses JavaScript to call REST APIs to retrieve and display items in the catalog and to place orders.
  • Node.js back end that uses the Express framework to implement REST APIs that return catalog details and place orders. Each REST API uses the invoke API to call COBOL programs from the CICS catalog manager.
  • CICS Node.js application profile is used by CICS to setup the Node.js runtime and environment variables passed to the application.
  • CICS bundle that contains the Node.js application and the profile and is used to install and manage the application in CICS.
